# nyc-yellow-trips-aws-lab

NYC Yellow Taxi Data – AWS S3 Hands-on Lab

## Objective
Practice core AWS + GitHub fundamentals using the NYC Yellow Taxi dataset:
- Create an S3 bucket and upload multiple file types (CSV, Parquet, etc.)
- Enable bucket versioning
- Apply lifecycle policies (Standard → IA → Glacier → Expire)
- Create IAM user, group, and role with S3 read-only access
- Practice GitHub workflow using branches and Pull Requests (PR)

## Dataset Links (Reference)
- Yellow Trips Data Dictionary
- Taxi Zone Lookup Table (CSV)
- Yellow Taxi Trip Records (PARQUET)

## Repo Structure
- `data/` : Dataset files (CSV/Parquet) used for the lab
- `docs/` : Screenshots + task notes for supervisor submission
